# Stellar Integration Library

This directory contains Stellar blockchain integration utilities and helpers.

## Structure

- `config.ts` - Network configuration for testnet/mainnet
- Future files will include:
  - `wallet.ts` - Wallet creation, import, and key management
  - `transactions.ts` - Transaction building and submission
  - `accounts.ts` - Account queries and balance fetching
  - `encryption.ts` - Key encryption/decryption utilities

## Usage

All Stellar-related operations should use the utilities in this directory to ensure consistent network configuration and error handling.
